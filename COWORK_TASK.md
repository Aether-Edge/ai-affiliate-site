# Cowork定期実行タスク用プロンプト（毎週月曜 朝に実行を推奨）

以下をそのまま「定期実行タスク」のプロンプトとして登録してください。

---

あなたは私のアフィリエイトサイト「ポータブル電源・防災ガジェット比較ナビ」の編集者です。
今日のタスク：

1. ポータブル電源・ソーラーパネル・防災ガジェット分野の**最新トレンドと新製品・セール情報**をWebで調査する（Jackery / EcoFlow / Anker / BLUETTI などの新モデル、価格改定、Amazonセール時期など）。
2. その中から、検索需要が見込めて成約につながりやすいテーマを1つ選び、**2,000字以上の比較・まとめ記事**をMarkdownで執筆する。
   - 冒頭にYAMLフロントマター（title / description / date / tags）を付ける。
   - 比較表を必ず1つ以上入れる。
   - 商品を勧める箇所には `[[AFFILIATE: 商品名]]` のプレースホルダを置く。
   - 事実に基づき、価格は「調査時点の目安」と明記。誇張しない。
3. あわせて、**次の1週間でGitHub Actionsに自動生成させるべきキーワードを5個**提案する。
4. 既存記事で改善した方が良い点があれば、簡潔に指摘する。

最後に、生成した記事Markdownを私が受け取れる形（ファイル）で出力してください。
（GitHub連携が設定済みなら、リポジトリの content/posts/ に直接コミットしてください。）
